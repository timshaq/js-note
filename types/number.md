# number
> может быть не больше 2\*\*53 - 1 и не менее -(2\*\*53 - 1).

> Для больших или меньших есть BigInt

## Бесконечность
> бесконечность появляется при делении на 0
```
console.log(1/0) // Infinity
console.log(-1/0) // -Infinity
```
## NaN
> вычислительная ошибка
```
console.log("asd"/2) // NaN
```
```
typeof NaN; // number
```
